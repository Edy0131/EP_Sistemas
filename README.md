# EP_Sistemas

Projeto EP_Sistemas — primeiro deploy realizado com sucesso.

## Como executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/Edy0131/EP_Sistemas.git
   cd EP_Sistemas
   ```

2. Abra o `index.html` no navegador ou use um servidor local.

## Estrutura do projeto

```
EP_Sistemas/
├── index.html       # Página principal
├── css/
│   └── style.css    # Estilos
├── js/
│   └── main.js      # Scripts
└── README.md
```

## Deploy

Repositório configurado e publicado no GitHub:

```bash
git init
git add .
git commit -m "primeiro deploy"
git branch -M main
git remote add origin https://github.com/Edy0131/EP_Sistemas.git
git push -u origin main
```
